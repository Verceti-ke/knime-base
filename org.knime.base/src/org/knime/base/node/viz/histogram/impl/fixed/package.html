This is the memory saving implementation of the abstract histogram plotter. The user has to specify the x and y axis in the node configuration and can change the following properties on the fly:
  1. Number of bars for numerical columns
  2. The size of the bars
  3. the background color
  4. Aggregation method
  5. Show/hide missing value bar
  6. Show/hide empty bars
This node is applicable for huge data sets since it only holds the neccessary data in the main memory.
