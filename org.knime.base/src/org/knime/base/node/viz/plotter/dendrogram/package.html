Contains all classes necessary to visualize a hierachical clustering result represented by a {@link org.knime.base.node.viz.plotter.dendrogram.ClusterNode}. The {@link org.knime.base.node.viz.plotter.dendrogram.DendrogramPlotter} converts the {@link org.knime.base.node.viz.plotter.dendrogram.ClusterNode} in a {@link org.knime.base.node.viz.plotter.dendrogram.BinaryTree} of {@link org.knime.base.node.viz.plotter.dendrogram.DendrogramPoint}s, which represent the visual model of the hierarchical clustering result. The {@link org.knime.base.node.viz.plotter.dendrogram.DendrogramDrawingPane} inteprets this tree into a dendrogram by adding the horizontal and vertical lines between the node and it's subnodes.
