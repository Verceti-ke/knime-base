This package contains convenient classes when two or n columns should be displayed. The MultiColumnProperties is a tab for filtering columns, TwoColumnProperties is a properties panel with the default properties (as defined in AbstractPlotterProperties) with an additional tab where two columns can be selected and the displayed ranges of the columns can be adjusted. The TwoColumnPlotter is a wrapper class which registeres all necessary listeners to the TwoColumnPropeties interaction components and calls the according methods on events.
